# MystaGram

Instagram clone with React, React Native, Express, and Prisma

## User Stories

- [x] Create account
- [ ] Request Secret
- [ ] Confirm Secret (Login)
- [ ] Like / Unlike a photo
- [ ] Comment on a photo
- [ ] Search by user
- [ ] Search by location
- [ ] See user profile
- [ ] Follow / Unfollow User
- [ ] See the full photo
- [ ] Edit my profile
- [ ] Upload a photo
- [ ] Edit the photo (Delete)
- [ ] See the feed
